Configuration Guide
Label Size Options:
  • Small: Minimal information, best for small label sheets
  • Medium: Standard information, good for most use cases
  • Large: Detailed information, best for large label sheets
Labels Per Row:
  • 2 Labels: Large labels, good for detailed printing
  • 3 Labels: Medium labels, balanced size
  • 4 Labels: Standard labels, most common
  • 6 Labels: Small labels, maximum efficiency
Global Settings: These settings apply to all label printing across the entire system. Individual modules will inherit these settings unless specifically overridden.
