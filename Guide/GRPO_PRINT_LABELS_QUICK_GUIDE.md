# GRPO Print Labels - Quick Guide

## How to Print Labels

### Step 1: Generate Labels
1. Go to GRPO Transfer session detail page
2. Click "QC Validation" tab
3. Fill in quantities and approve items
4. Click "Submit QC Approval"
5. Go to "QR Labels" tab
6. Click "Generate Labels" button
7. Confirm generation

### Step 2: View Labels
1. Labels automatically display in the QR Labels tab
2. Each label shows:
   - QR code
   - Item code and name
   - Batch number
   - Warehouse information
   - Date

### Step 3: Print Labels
1. Click "Print All Labels" button
2. New window opens with all labels
3. Click "Print Labels" button in the new window
4. Select printer and settings
5. Click "Print"

---

## Label Information Displayed

Each label shows:
- **PO**: Purchase Order / Session Code
- **Item Code**: Item identifier
- **Item Name**: Item description
- **Batch**: Batch number
- **Qty per Pack**: Quantity per label
- **From Warehouse**: Source warehouse
- **To Warehouse**: Destination warehouse
- **GRN Date**: Date of receipt
- **Pack**: Label number (e.g., "Pack 1 of 4")

---

## Print Layout

- **Format**: 2 columns per page
- **Paper**: A4 or Letter size
- **QR Code Size**: 200x200 pixels
- **Quality**: 150+ DPI recommended

## Troubleshooting

### Labels Not Showing
- Verify labels were generated
- Check if QC approval was submitted
- Refresh the page

### Print Window Not Opening
- Check popup blocker
- Enable JavaScript
- Try different browser

### QR Codes Not Printing
- Check internet connection
- Verify printer settings
- Try different printer

---

## Tips

1. **Best Print Quality**: Use 300 DPI setting
2. **Paper Size**: Use A4 or Letter size
3. **Margins**: Set to 10mm or less
4. **Color**: Print in color for better QR code scanning
5. **Preview**: Use print preview before printing

---

## Keyboard Shortcuts

- **Print**: Ctrl+P (or Cmd+P on Mac)
- **Close Print Window**: Alt+F4 (or Cmd+W on Mac)
